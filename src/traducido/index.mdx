import { getCommunity } from "./community";# Bienvenido a Qwik

Qwik es un nuevo tipo de marco web que puede ofrecer aplicaciones web de carga instantánea a cualquier tamaño o complejidad. Sus sitios y aplicaciones pueden arrancar con aproximadamente 1kb de JS (independientemente de la complejidad de la aplicación), y lograr un rendimiento consistente a escala.
